# E-Commerce Customer Behavior & Sales Analysis (Power BI)

## Overview
This project presents an interactive Power BI dashboard built to analyze e-commerce customer behavior and sales performance.

The dashboard focuses on understanding revenue distribution, customer retention patterns, payment method preferences, device usage, and sales trends over time.

## Key Insights
- Electronics contributes the highest share of total revenue
- Credit card users form the largest segment of returning customers
- Mobile devices dominate session duration
- Sales peak around mid-year and dip in early 2024

## Dashboard Features
- Revenue by Product Category (Treemap)
- Returning Customers by Payment Method
- Pages Viewed by Payment Method
- Session Duration Breakdown by Device Type
- Monthly Sales Volume Trend
- Interactive slicers for Gender, Product Category, Device Type, and Delivery Time

## Tools Used
- Power BI Desktop
- DAX (for KPI calculations)
- Data modeling and interactive filtering

## Dataset
Dataset sourced from Kaggle:  
E-Commerce Customer Behavior & Sales Analysis

## How to View
- Download the `.pbix` file from the `dashboard` folder
- Open it using Power BI Desktop

## Notes
This dashboard is filter-aware, and KPIs update dynamically based on slicer selections.
